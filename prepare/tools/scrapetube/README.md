## Hướng dẫn sử dụng

- Chạy file `crawl.py` để thu thập toàn bộ video list của toàn bộ danh sách kênh đã thu thập trước đó.

- Chạy file `update_res.py` để mở rộng file `res.csv` sau đó đổi tên file vừa tạo thành `video_urls.csv` để sử dụng cho bước tiếp theo.

- scrapetube là một thư viện cho phép thu thập thông tin danh sách video cả ba dạng gồm `static video (video)`, `short` và `live stream` của một kênh YouTube.
